# 수치 관련 정리 사항

에디터 Inspector에서 조정 가능한 모든 수치와 영향 범위를 정리합니다.

---

## 전차 하드웨어 (ChariotCrewController)

> 하이어라키: `Battle > Chariot` (ChariotCrewController 컴포넌트)

| 변수명 | 타입 | 기본값 | 설명 | 영향 |
|--------|------|--------|------|------|
| horsePullCapacity | float | 800 | 말 견인력 (최대 적재량) | 총 무게 / 견인력 = 하중비. 초과 시 속도 급감 (최대 -80%) |
| horseBaseMoveSpeed | float | 12 | 말 기본 이동속도 | 최종속도 = 기본속도 × 하중패널티 × (1 + 마부숙련 × 0.02) |
| horseWeight | float | 400 | 말 무게 | 총 무게에 합산. 무거울수록 속도↓, 충돌데미지↑ |
| armorDurability | float | 500 | 갑옷 내구도 | ChariotStats.maxHP로 적용. 전차 최대 HP |
| armorDefense | float | 30 | 갑옷 방어력 | 피격 시 데미지 차감: finalDmg = max(1, dmg - defense) |
| armorWeight | float | 300 | 갑옷 무게 | 총 무게에 합산 |
| wheelAccel | float | 5 | 바퀴 가속도 | 가속 성능 (추후 구현) |
| wheelCollisionDamage | float | 50 | 바퀴 충돌 데미지 | 전차 충돌 시 기본 데미지. 무게 비례 보정 적용 (미연결) |
| wheelWeight | float | 100 | 바퀴 무게 | 총 무게에 합산 |

### 무게/속도 공식
```
총 무게 = 말 + 갑옷 + 바퀴 + 궁병(체중+활) + 창병(체중+창) + 검병(체중+검) + 마부(체중)

하중비 = 총 무게 / 견인력
  ≤ 1.0 → 속도 소폭 감소 (최대 -20%)
  > 1.0 → 급격 감소 (최대 -80%)

최종 속도 = 기본속도 × 하중패널티 × (1 + 마부숙련 × 0.02)
```

## 궁병 (ArcherView)

> 플레이어: `Battle > Chariot > Archer`
> 적 전차: `EnemyChariot > Archer`

| 변수명 | 타입 | 기본값 | 설명 | 영향 |
|--------|------|--------|------|------|
| level | int | 1 | 초기 레벨 | 모든 스탯에 레벨 배율 적용 (공격 ×1.05/Lv, 방어 ×1.04/Lv, HP ×1.06/Lv) |
| archerySkill | float | 0 | 궁술 숙련 | 사거리·데미지에 (1 + skill × 0.01) 배율. 레벨업 시 +0.15 |
| bowRange | float | 30 | 활 기본 사거리 | 궁병 공격 권역 범위 결정. 실효사거리 = bowRange × (1 + skill × 0.01) |
| bowBasePower | float | 0 | 활 기본 위력 | WeaponSpec에 전달 (현재 데미지 계산에 미사용) |
| bowWeight | float | 5 | 활 무게 | 전차 총 무게에 합산 |
| bodyWeight | float | 70 | 궁병 체중 | 전차 총 무게에 합산 |
| baseAttack | float | 18 | 기본 공격력 | 데미지 = baseAttack × 레벨배율 × (1 + skill × 0.01) |
| baseDefense | float | 3 | 기본 방어력 | 개별 HP 미구현 (추후용) |
| baseMaxHp | float | 90 | 기본 최대 HP | 개별 HP 미구현 (추후용) |
| attackInterval | float | 1.2 | 공격 쿨타임 (초) | 낮을수록 DPS↑. 1.2초마다 화살 1발 |
| arrowPool | ObjectPool | null | 화살 풀 참조 | null이면 화살 발사 불가. 플레이어는 직접 소유, 적은 런타임 주입 |

### 궁병 DPS 공식
```
데미지 = baseAttack × (1 + (level-1) × 0.05) × (1 + archerySkill × 0.01)
DPS = 데미지 / attackInterval
```

| meleeRange | float | 2.0 | 근거리/원거리 경계 | 이 거리 이하 → 즉시 찌르기, 초과 → 투창 발사 |
| spearPool | ObjectPool | null | 투창 풀 참조 | null이면 투창 불가, 근거리만 가능 |

### 창병 DPS 공식
```
데미지 = baseAttack × (1 + (level-1) × 0.05) × (1 + lanceSkill × 0.01)
DPS = 데미지 / attackInterval

근거리(≤ meleeRange): 즉시 찌르기 데미지
원거리(> meleeRange): 투창 Projectile 발사 (spreadX 분산 적용)
```

## 검병 (SwordsmanView)

> 플레이어: `Battle > Chariot > Swordsman`
> 적 전차: `EnemyChariot > Swordsman`

| 변수명 | 타입 | 기본값 | 설명 | 영향 |
|--------|------|--------|------|------|
| level | int | 1 | 초기 레벨 | 동일 레벨 배율 |
| swordSkill | float | 0 | 검술 숙련 | 사거리 (1 + skill × 0.005), **데미지 (1 + skill × 0.04)**. 레벨업 시 +0.12. 데미지 스케일링이 궁병/창병의 4배 |
| swordRange | float | 1.5 | 검 사거리 | 가장 짧은 권역. 검병 공격 권역 = 0 ~ swordRange |
| swordBasePower | float | 20 | 검 기본 위력 | **검병 고유**. 공격력에 가산: (baseAttack + swordBasePower) |
| swordWeight | float | 6 | 검 무게 | 전차 총 무게에 합산 |
| bodyWeight | float | 78 | 검병 체중 | 전차 총 무게에 합산 |
| baseAttack | float | 22 | 기본 공격력 | 승무원 중 가장 높음 |
| baseDefense | float | 4 | 기본 방어력 | 개별 HP 미구현 |
| baseMaxHp | float | 100 | 기본 최대 HP | 개별 HP 미구현 |
| attackInterval | float | 0.6 | 공격 쿨타임 (초) | 가장 빠름. 0.6초마다 베기 |

### 검병 DPS 공식
```
데미지 = (baseAttack + swordBasePower) × (1 + (level-1) × 0.05) × (1 + swordSkill × 0.04)
DPS = (22 + 20) × 레벨배율 × 스킬배율 / 0.6
→ 승무원 중 최고 DPS (스킬 스케일링 4배)
```

## 마부 (CoachmanView)

> 플레이어: `Battle > Chariot > Coachman`
> 적 전차: `EnemyChariot > Coachman` (있을 경우)

| 변수명 | 타입 | 기본값 | 설명 | 영향 |

| handlingSkill | float | 0 | 마차 조종 숙련 | 이동속도 × (1 + skill × 0.02), 회피 = skill × 0.003. 레벨업 시 +0.2 |
| bodyWeight | float | 75 | 마부 체중 | 전차 총 무게에 합산 |
| baseAttack | float | 5 | 기본 공격력 | 미사용 (전투 참여 안 함) |
| baseDefense | float | 8 | 기본 방어력 | 승무원 중 가장 높은 방어 |
| baseMaxHp | float | 110 | 기본 최대 HP | 승무원 중 가장 높은 HP |

### 마부 영향
```
skill 0  → 속도 보정 ×1.00
skill 10 → 속도 보정 ×1.20 (+20%)
skill 50 → 속도 보정 ×2.00 (+100%)
→ 고레벨 마부는 무거운 장갑 전차의 속도 페널티를 상쇄 가능
```

## 승무원 DPS 비교 (Lv1 기준)

| 병종 | 데미지 | 쿨타임 | DPS | 스킬 스케일링 |
|------|--------|--------|-----|---------------|
| 검병 | (22+20) × 1.0 = 42 | 0.6초 | **70.0** | ×0.04 (최고) |
| 창병 | 16 × 1.0 = 16 | 1.0초 | 16.0 | ×0.01 |
| 궁병 | 18 × 1.0 = 18 | 1.2초 | 15.0 | ×0.01 |

> 검병이 Lv1부터 DPS 4배 이상. 스킬 스케일링도 4배여서 후반으로 갈수록 격차 벌어짐.
