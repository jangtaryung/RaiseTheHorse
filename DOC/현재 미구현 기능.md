# 현재 미구현 기능 목록

> 기획서 "전차 키우기 기획 정리 중.txt" 기준으로 실제 구현과의 차이를 정리.
> 기획서에서 반복적으로 강조되거나 핵심 재미와 직결되는 항목을 높은 우선순위로 분류.
> 작성일: 2026-03-23

## P0 — 기획 핵심 재미에 직결 (게임 루프 자체가 달라지는 항목)

### 1. 장비 교체/파츠 조합 시스템
- **기획서 근거**: 2.3, 3.2, 3.3 — 기획서 전체에서 가장 많이 반복되는 핵심 시스템
- **기획 의도**: 견인력 한도 내에서 장갑/바퀴/무장 파츠를 교체하며 트레이드오프 발생
  - 예: "무거운 대형 활 장착 → 장갑을 얇은 것으로 교체"
- **현재 상태**: 파츠 고정, 교체 불가. 업그레이드는 순수 수치 상승만 있어 포기/선택이 없음
- **영향**: "어떤 전차로 키울 것인가"라는 핵심 철학(1.2)이 작동하지 않음
- **구현 범위**:
  - [ ] 파츠별 여러 등급/종류 데이터 구조
  - [ ] 장착/교체 UI
  - [ ] 견인력 초과 시 장착 제한 로직
  - [ ] 파츠 교체에 따른 무게 재계산

### 2. 스테이지 목표/도달 시스템 (전술적 레이스)
- **기획서 근거**: 4.1, 4.2 — 메인 컨텐츠의 근간
- **기획 의도**: 각 스테이지에 목적지(성문, 요충지)가 있고, 도달 시간에 따라 차등 보상
  - "속도 vs 안정성" 전략적 선택 유발
- **현재 상태**: 목적지 없음. 무한 스폰 서바이벌 형태. 죽으면 게임오버
- **영향**: 게임 루프가 "전술적 레이스"가 아닌 "생존 디펜스"로 완전히 다름
- **구현 범위**:
  - [ ] 스테이지별 목적지(골인 지점) 설정
  - [ ] 도달 시간 측정
  - [ ] 시간 기반 차등 보상
  - [ ] 스테이지 클리어/다음 스테이지 전환

### 3. 재화/자원 시스템
- **기획서 근거**: 1.2, 2.2, 3.4, 4.3 — 육성 동기의 기반
- **기획 의도**: 전투로 금화/자원 획득 → 한정된 자원을 어디에 투자할지 선택
- **현재 상태**: 경험치만 존재. 레벨업 시 무료 랜덤 카드 선택
- **영향**: "선택과 집중"(3.3) 재미가 자원 부재로 운 요소에 의존
- **구현 범위**:
  - [ ] 재화 종류 정의 (금화, 희귀 자원 등)
  - [ ] 전투 보상으로 재화 획득
  - [ ] 재화 소모로 업그레이드/파츠 구매
  - [ ] 타임 어택 보상 배율

## P1 — 전투 체감에 직접 영향 (기획서 키피쳐에 명시된 항목)

### 4. 창병 넉백/저지력
- **기획서 근거**: 2.1, 3.1 — 창병의 정체성 자체
- **기획 의도**: 창병이 적을 "밀쳐내어 접근을 차단"하는 중거리 방어선
- **현재 상태**: 데미지만 적용, 적 위치를 밀어내는 물리 반응 없음
- **영향**: 창병이 "느린 검병"처럼 느껴짐. 병과 간 역할 차이 약화
- **구현 범위**:
  - [ ] 창병 공격 적중 시 넉백 벡터 적용
  - [ ] LanceSkill에 따른 넉백 거리 스케일링
  - [ ] 적 이동 로직에 넉백 처리 반영

### 5. 회피(Evasion) 판정 연결
- **기획서 근거**: 3.1 — 마부 역할 "전차의 기동 효율 및 회피 판정"
- **기획 의도**: 마부 숙련도가 높을수록 회피율 상승, 무게 초과 시 페널티
- **현재 상태**: `GetCurrentEvasion()` 공식 완비되어 있으나 `TakeDamage()`에서 한 번도 호출 안 함
- **영향**: 마부 육성의 가치가 이동속도에만 한정됨. 기획상 핵심 역할 절반 미작동
- **구현 범위**:
  - [ ] `TakeDamage()` 진입 시 `GetCurrentEvasion()` 확률로 데미지 무효화
  - [ ] 회피 발생 시 피드백 (이펙트/로그)

### 6. 궁병 스킬 → 명중률/재장전 연동
- **기획서 근거**: 3.1 — "궁술 숙련 → 재장전 속도 및 명중률"
- **기획 의도**: 궁술 숙련이 오를수록 화살이 정확해지고 빠르게 재장전
- **현재 상태**: ArcherySkill은 사거리와 데미지에만 영향. `spreadX`(착탄 분산)는 프리팹 고정값 1.5, `attackInterval`도 고정값 1.2
- **영향**: 궁병 숙련을 올려도 체감 변화가 데미지뿐. 기획서 명시 효과 2개 미작동
- **구현 범위**:
  - [ ] ArcherySkill → spreadX 감소 (명중률 향상)
  - [ ] ArcherySkill → attackInterval 감소 (재장전 속도 향상)

### 7. 충돌 데미지 LayerMask 확인/정비
- **기획서 근거**: 2.4 — "전차 본체의 충돌 판정으로 적진을 뚫고 지나가는 기동 액션"
- **기획 의도**: 전차가 적에게 돌진하여 물리적 데미지
- **현재 상태**: 코드 연결은 되어 있으나 `chariotCollisionLayers`가 Inspector에서 미설정이면 작동 안 함 (설정 의존 상태)
- **영향**: 키피쳐(2.4)인데 실제 플레이에서 작동하는지 불확실
- **구현 범위**:
  - [ ] chariotCollisionLayers 설정 확인 및 적용
  - [ ] 충돌 시 시각적 피드백 (이펙트/화면 흔들림)

## P2 — 육성 깊이/진행감 (장기 플레이 루프)

### 8. 숙련도(Skill) 직접 투자
- **기획서 근거**: 2.2, 3.1 — "어느 병과의 화력을 우선시할 것인가"
- **기획 의도**: 유저가 숙련도 배분을 직접 결정하여 빌드 분화
- **현재 상태**: 스킬은 레벨업 시 자동 증가(0.12~0.2/레벨)만 존재. 유저 선택 불가
- **영향**: 병과 특화 빌드를 의도적으로 만들 수 없음
- **구현 범위**:
  - [ ] 업그레이드 카드에 스킬 계열 추가 (또는 별도 스킬 투자 UI)
  - [ ] ArcherySkill, LanceSkill, SwordSkill, ChariotHandlingSkill 직접 투자 경로

### 9. 검병 스킬 → 공격속도 연동
- **기획서 근거**: 3.1 — "검술 숙련 → 초근접 공격 속도 및 살상력"
- **현재 상태**: SwordSkill은 데미지와 사거리에만 영향. `attackInterval` 고정 0.6
- **구현 범위**:
  - [ ] SwordSkill → attackInterval 감소 (공격속도 향상)

### 10. 난이도 진행/스케일링
- **기획서 근거**: 3.3, 4.2 — "다음 스테이지(더 높은 견인력과 속도 요구)에 도전"
- **현재 상태**: BattleSpawner 고정 간격(2초), 적 스펙 고정
- **구현 범위**:
  - [ ] 스테이지별 적 스펙 상승
  - [ ] 스폰 간격/밀도 조절
  - [ ] 적 구성 다양화 (근접형, 원거리형 등)

### 11. 성능의 시각화/피드백
- **기획서 근거**: 3.4 — "투사체의 궤적, 적의 밀려남 정도 등으로 즉각 반영"
- **현재 상태**: 수치 변화는 내부적으로 적용되지만 시각적 피드백 부족
- **구현 범위**:
  - [ ] 스킬/스탯 변화에 따른 시각 변화 (화살 속도, 궤적 변화 등)
  - [ ] 데미지 숫자 표시
  - [ ] 회피/넉백 발생 시 이펙트
